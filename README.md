# Select Option Composite Products

Este plugin estende o [WPC Composite Products para WooCommerce](https://wordpress.org/plugins/woo-composite-products/) adicionando novas opções para controlar o comportamento dos componentes.

## Funcionalidades

- **Selecionar automaticamente todos os produtos dos componentes:** habilite para que todos os itens disponíveis em cada componente sejam escolhidos assim que a página do produto for carregada.
- **Ocultar o botão "Escolher":** remova o link `wooco_component_product_selection_list_item_choose` da interface para impedir que os clientes alterem manualmente a seleção.

## Configuração

1. Acesse **WooCommerce → Composite Products Options** no painel administrativo.
2. Marque as opções desejadas e salve.

> ⚠️ Este plugin depende do **WPC Composite Products para WooCommerce**. Certifique-se de que ele esteja instalado e ativo.
